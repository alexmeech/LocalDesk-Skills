# LocalDesk Skills Marketplace

## Обзор

Это веб-маркетплейс проверенных навыков (skills) для Claude AI, оптимизированных для использования в LocalDesk.

## Возможности

- **Динамическая загрузка**: Все навыки загружаются автоматически через GitHub API
- **Поиск**: Быстрый поиск по названию, описанию и категории
- **Фильтрация**: Удобная фильтрация по категориям
- **Статистика**: Отображение общего количества навыков и категорий
- **Прямые ссылки**: Каждый навык содержит прямую ссылку на репозиторий GitHub

## Как это работает

1. При загрузке страницы JavaScript код обращается к GitHub API
2. Получает список всех папок в директории `skills/`
3. Для каждой папки загружает файл `SKILL.md`
4. Парсит YAML frontmatter для извлечения метаданных:
   - `name` - название навыка
   - `description` - описание
   - `license` - лицензия
5. Автоматически категоризирует навыки
6. Отображает их в виде красивых карточек

## Добавление нового навыка

Чтобы добавить новый навык в маркетплейс:

1. Создайте новую папку в `skills/`
2. Добавьте файл `SKILL.md` с YAML frontmatter:
```markdown
---
name: my-skill-name
description: Описание вашего навыка
license: Apache 2.0
---

# Содержимое навыка
...
```

3. (Опционально) Добавьте категорию в `categoryMap` в `docs/index.html`
4. Коммитьте и пушьте - навык автоматически появится на сайте!

## Категории

- **Креатив**: Алгоритмическое искусство и креативные инструменты
- **Дизайн**: Дизайн интерфейсов, брендинг, канвас-дизайн
- **Разработка**: Веб-разработка, тестирование, MCP серверы
- **Документы**: Работа с DOCX, PDF, PPTX, XLSX
- **Коммуникация**: Внутренние коммуникации, Slack-интеграции
- **Утилиты**: Вспомогательные инструменты

## Технологии

- Чистый HTML/CSS/JavaScript (без фреймворков)
- GitHub API для динамической загрузки
- GitHub Pages для хостинга
- Anthropic брендинг (цвета и шрифты)
